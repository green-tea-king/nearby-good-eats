綜合評價怎麼算？
① 貝氏加權分數
分數 ＝ ( C × m ＋ n × r ) ÷ ( C ＋ n )

r＝Google 評分，n＝評論數，m＝當批平均分，C＝150（可信度門檻）。評論數少的高分店會被拉回平均，「評論多又高分」才會浮上來，避免「5.0 分但只有 3 則」灌水。

② 評鑑加分
  • 米其林　三星 +0.5・二星 +0.4・一星 +0.3
  • 必比登推介　+0.2
  • 500 盤　每盤 +0.03（上限 +0.4）
綜合評價 ＝ 貝氏分數 ＋ 評鑑加分
